title: OS Login Should Be Enabled
---

### More Info:

Enable OS login to ensure that SSH keys used to connect to instances are mapped with IAM users.

### Risk Level

High

### Address

Security

### Compliance Standards

CIS, SOC2, PCIDSS, NIST, HIPAA, ISO27001
