### More Info:

It is recommended that the IAM policy on BigQuery datasets does not allow anonymous and/or public access.

### Risk Level

Medium

### Address

Security

### Compliance Standards

CISGCP, CBP, HITRUST, GDPR, SOC2, NISTCSF, PCIDSS, FedRAMP
