### More Info:

Ensure that cloud Storage buckets do not allow All Users to have Ownership ("allUsers" must not have "OWNER" roles)

### Risk Level

Critical

### Address

Security

### Compliance Standards

HITRUST, SOC2, NISTCSF, PCIDSS
