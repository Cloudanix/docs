title: GCP Cloud Function Should Not Have Admin Access
---

### More Info:

Ensure that the Cloud Function users do not have administrative priveleges

### Risk Level

High

### Address

Security

### Compliance Standards

CBP, PCIDSS, HIPAA, APRA, MAS, NIST

### Remediation

How to disable Admin access for GCP Function.

#### Using GCP Console

1. Log in to the Google Cloud Platform (GCP) Console using your GCP account credentials.
2. Navigate to the Cloud Functions section by selecting "Cloud Functions" from the main menu.
3. Select the Cloud Function for which you want to remediate the admin access.
4. In the Cloud Function details page, click on the "Edit" button.
5. Scroll down to the "Cloud Function Permissions" section and locate the "Member" field with the admin role assigned.
6. Identify the member or service account that has been granted the admin role access.
7. Click on the "X" icon or remove button next to the member's entry to remove the admin access.
8. Review the permissions and roles assigned to the Cloud Function and ensure that only necessary and appropriate roles are assigned.
9. Click on the "Save" button to apply the changes to the Cloud Function.
10. Repeat these steps for any other Cloud Functions that have admin access assigned to unauthorized members.
