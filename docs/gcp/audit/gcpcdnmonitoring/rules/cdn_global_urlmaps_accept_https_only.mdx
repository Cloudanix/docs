### More Info:

Cloud CDN global url maps should be configured to block HTTP connection and allow only HTTPS connections.

### Risk Level

High

### Address

Security

### Compliance Standards

GDPR, PCIDSS, NIST, HITRUST, NISTCSF, SOC2
