### More Info:

Ensure that no network security groups allow unrestricted inbound access on TCP port 139 and UDP ports 137 and 138 (NetBIOS).

### Risk Level

High

### Address

Security

### Compliance Standards

HIPAA, NIST, HITRUST, GDPR, SOC2, NISTCSF, PCIDSS, FedRAMP
