### Remediation

How to remediate an AWS CloudFormation stack configuration that has been drifted?

#### Using AWS Console

1. Select the CloudFormation stack which is drifted.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "AWS CloudFormation Drift Detection" Policy.)
2. From the Dashboard top menu, click the Update button.
3. Choose Use current template to update the stack using the CloudFormation template with the expected configuration.
4. Ensure that Stack parameters are correct on the Specify stack details step.
5. Follow the instructions to finish and you should see UPDATE COMPLETE.
