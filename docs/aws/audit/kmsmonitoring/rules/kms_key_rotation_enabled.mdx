### More Info:

When you enable automatic key rotation, AWS KMS rotates the CMK 365 days after the enable date and every 365 days thereafter.

### Remediation

How to enable KMS Key rotation?

#### Using AWS Console

1. Open the AWS Management Console and navigate to the KMS service.
2. Select the KMS key for which you want to enable key rotation.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "KMS Key Rotation Should Be Enabled" Policy.)
3. Click on the "Key policy" button.
4. In the key policy editor, add the following statement to the key policy:

```
{
  "Sid": "Enable key rotation",
  "Effect": "Allow",
  "Principal": {
    "AWS": "*"
  },
  "Action": [
    "kms:EnableKeyRotation",
    "kms:UpdatePrimaryRegion"
  ],
  "Resource": "*"
}
```

5. Save the changes to the key policy.
6. Navigate to the "Key rotation" tab and click on the "Enable" button.
7. Set the key rotation interval to your desired frequency (e.g. yearly, bi-annually, etc.).
8. Click on the "Save changes" button.
