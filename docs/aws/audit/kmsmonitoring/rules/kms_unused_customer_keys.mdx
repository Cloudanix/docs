### More Info:

Any disabled KMS Customer Master Keys in your AWS account should be removed in order to lower the cost of your monthly AWS bill.

### Remediation

How to remove unused Customer Master Key?

#### Using AWS Console

1. Open the AWS Key Management Service (KMS) console.
2. In the navigation pane, choose "Customer managed keys".
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "Unused Customer Master Key Should Be Removed" Policy.)
3. Select the CMK you want to remove.
4. In the "Key actions" drop-down menu, choose "Schedule key deletion".
5. In the "Schedule key deletion" dialog box, enter the number of days you want to keep the key before deletion. The default is 30 days.
6. Choose "Schedule deletion".
