title: Event Bus Should Not Be Exposed
---

### More Info:

Your AWS CloudWatch event bus should not be exposed to everyone.

### Risk Level

High

### Address

Security

### Compliance Standards

PCIDSS, GDPR, APRA, MAS, NIST
