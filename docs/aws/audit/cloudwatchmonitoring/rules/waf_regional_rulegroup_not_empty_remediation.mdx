2. List all the WAF Regional Rule Groups: Use the following AWS CLI command to list all the WAF Regional Rule Groups in your account:

   ```
   aws wafv2 list-rule-groups --scope REGIONAL
   ```

   This command will return a list of all the WAF Regional Rule Groups in your account.

3. Check each Rule Group for rules: For each Rule Group returned in the previous step, use the following AWS CLI command to get the details of the Rule Group:

   ```
   aws wafv2 get-rule-group --scope REGIONAL --id <RuleGroupId> --name <RuleGroupName> --version <RuleGroupVersion>
   ```

   Replace `<RuleGroupId>`, `<RuleGroupName>`, and `<RuleGroupVersion>` with the ID, name, and version of the Rule Group you want to check. This command will return the details of the Rule Group, including the rules it contains.

4. Check if the Rule Group is empty: In the output of the previous command, look for the "Rules" field. If this field is empty, then the Rule Group is empty. If it contains one or more rules, then the Rule Group is not empty.
