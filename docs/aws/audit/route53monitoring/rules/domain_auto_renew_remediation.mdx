### Remediation

How to enable AWS Route 53 Auto Renew?

#### Using AWS Console

1. Log in to the AWS Management Console using your AWS account credentials.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "AWS Route 53 Auto Renew Should Be Enabled" Policy.)
2. Navigate to the Route 53 service by selecting "Route 53" from the services menu.
3. In the Route 53 dashboard, click on "Registered domains" in the left navigation pane.
4. Select the domain for which you want to enable autorenewal.
5. In the domain details page, click on the "Add or edit name servers" button.
6. Ensure that the domain is using the Route 53 name servers. If it's not, follow the instructions provided to update the name servers to Route 53.
7. Once the domain is using Route 53 name servers, go back to the domain details page.
8. In the domain details page, click on the "Auto renew" button.
9. In the "Enable auto renew" dialog box, review the information and click on the "Enable" button to enable autorenewal for the domain.
10. Verify that autorenewal is enabled by checking the status of the autorenewal setting in the domain details page.
11. Repeat these steps for any other domains in Route 53 that require autorenewal to be enabled.
