---
title: Root Account Should Have Password Rotation
---

### More Info:

Ensure that your root account password is rotated every few days.

### Risk Level

High

### Address

Security

### Compliance Standards

CBP, NIST

### Remediation

How to enable root account Password Rotation?

As of September 2021, there is no specific setting in AWS that allows you to disable password rotation for the AWS root account. However, it is recommended to regularly change your password to enhance the security of your AWS environment.
