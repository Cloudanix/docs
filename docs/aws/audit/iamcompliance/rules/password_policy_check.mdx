### More Info:

Password policy should be complex enough so that users can set passwords which are not easy to guess and crack.

### Risk Level

Medium

### Address

Security

### Compliance Standards

HIPAA, ISO27001, PCIDSS, GDPR, CISAWS, CBP, NIST, HITRUST, AWSWAF, SOC2, NISTCSF

### Remediation

How to ensure there is a complex password policy.

#### Using AWS Console

1. Sign in to the AWS Management Console using an account with administrative privileges.
2. Open the IAM (Identity and Access Management) console by navigating to the IAM service.
3. In the IAM console, click on "Account settings" in the navigation pane.
4. Click on the "Password policy" tab to view and configure the password policy settings.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "Complex Password Policy Should Be Present" Policy.)
5. Review the existing password policy, if any, and ensure that it meets your organization's requirements for complexity.
6. To edit the password policy, click on the "Edit" button.
7. Configure the following settings based on your organization's password policy requirements:
   - Minimum password length: Specify the minimum number of characters required for passwords.
   - Require at least one uppercase letter: Enable this option if uppercase letters are required in passwords.
   - Require at least one lowercase letter: Enable this option if lowercase letters are required in passwords.
   - Require at least one symbol: Enable this option if symbols (e.g., @, #, $) are required in passwords.
   - Require at least one number: Enable this option if numbers are required in passwords.
   - Allow users to change their own password: Enable this option to allow users to change their passwords.
   - Prevent password reuse: Specify the number of previous passwords that cannot be reused.
8. Optionally, you can set a password expiration period by enabling the "Enable password expiration" option and specifying the number of days.
9. Once you have configured the password policy settings, click on the "Apply password policy" button to save the changes.
