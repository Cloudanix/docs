### Remediation

How to enable HTTPS for CloudFront Distributions?

#### Using AWS Console

1. Open the AWS Management Console and navigate to the CloudFront service.
2. Select the CloudFront distribution for which you want to enable HTTPS.
   (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "HTTPS Should Be Enabled on CloudFront Distributions" Policy.)
3. Click on the "Distribution Settings" tab.
4. In the "General" section, click on the "Edit" button.
5. Scroll down to the "Viewer Protocol Policy" section and select the option "Redirect HTTP to HTTPS" or "HTTPS Only". Choose the appropriate option based on your requirements.
6. Save the changes by clicking on the "Yes, Edit" button.
