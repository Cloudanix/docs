---
title: ELBs Should Have Connection Draining Enabled
---

### More Info:

Elastic Load Balancer should not send any new requests to the unhealthy instance if an EC2 backend instance fails health checks

### Risk Level

Medium

### Address

Reliability

### Compliance Standards

CBP
