---
title: Unrestricted SSH Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access to TCP port 22 (SSH).

### Risk Level

Medium

### Address

Security

### Compliance Standards

CIS, SOC2, PCIDSS, NIST, HIPAA, ISO27001
