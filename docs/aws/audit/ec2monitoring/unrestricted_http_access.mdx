---
title: Unrestricted HTTP Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access to TCP port 80 (HTTP).

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, SOC2, GDPR, NIST, CIS, ISO27001, HIPAA
