---
title: Unrestricted Elasticsearch Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access to TCP port 9200 (Elasticsearch).

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, CBP, APRA, MAS, NIST
