---
title: Unrestricted RDP Access Should Not Be Allowed
---

### More Info:

No AWS EC2 security group should allow unrestricted inbound access to TCP port 3389 (RDP).

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, SOC2, NIST, CIS, ISO27001, HIPAA
