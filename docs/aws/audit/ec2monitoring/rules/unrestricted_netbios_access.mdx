### More Info:

No AWS EC2 security group should allow unrestricted inbound access to TCP port 139 and UDP ports 137 and 138 (NetBIOS).

### Risk Level

High

### Address

Security

### Compliance Standards

HIPAA, NIST, SOC2, GDPR
