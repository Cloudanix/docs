---
title: Unrestricted ICMP Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access using Internet Control Message Protocol (ICMP).

### Risk Level

Low

### Address

Security

### Compliance Standards

PCIDSS, APRA, MAS, NIST
