---
title: Unrestricted MongoDB Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted ingress access to MongoDB port 27017.

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, CBP, APRA, MAS, NIST
