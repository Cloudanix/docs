### Check Cause

#### Using Console

1. Log in to the AWS Management Console and navigate to the EC2 dashboard.
2. In the navigation pane, under "Instances", click on "Instances" to view all the instances.
3. Select the instance you want to check for the specified applications. Click on the "Actions" button, then click on "Instance Settings", and finally click on "Get System Log".
4. In the system log, you can check for the presence of the specified applications. The system log provides a log of the boot process of the instance, and any applications installed on the instance should be listed here.

#### Using CLI

1. First, you need to install and configure AWS CLI on your local machine. You can download it from the official AWS website and configure it using the "aws configure" command. You will be prompted to provide your AWS Access Key ID, Secret Access Key, Default region name, and Default output format.

2. Once the AWS CLI is set up, you can list all your EC2 instances using the following command:
   ```
   aws ec2 describe-instances
   ```
   This command will return a JSON output with the details of all your EC2 instances.

3. To check if a specific application is installed on an instance, you would need to SSH into the instance. You can do this using the following command:
   ```
   ssh -i /path/my-key-pair.pem my-instance-user-name@my-instance-public-dns-name
   ```
   Replace "/path/my-key-pair.pem" with the path to your private key file, "my-instance-user-name" with the name of the user account on the instance, and "my-instance-public-dns-name" with the public DNS name of the instance.

4. Once you are logged into the instance, you can check if a specific application is installed by using the appropriate command for the operating system. For example, on a Linux instance, you can use the following command to check if a package is installed:
   ```
   dpkg -l | grep package-name
   ```
   Replace "package-name" with the name of the application you want to check. If the application is installed, this command will return a line with the details of the installed package. If the application is not installed, this command will not return anything.

#### Using Python

1. **Setup AWS SDK (Boto3):**
   First, you need to set up AWS SDK (Boto3) for Python. This allows Python to interact with AWS services. You can install it using pip:
   ```
   pip install boto3
   ```
   Then, configure your AWS credentials either by setting up environment variables or by using the AWS CLI.

2. **List all EC2 Instances:**
   Use the `describe_instances()` function to get a list of all EC2 instances. Here is a sample script:
   ```python
   import boto3

   def list_instances():
       ec2 = boto3.resource('ec2')
       for instance in ec2.instances.all():
           print('ID: {}, State: {}, Type: {}'.format(instance.id, instance.state, instance.instance_type))

   list_instances()
   ```
   This script will print the ID, state, and type of all EC2 instances.

3. **Check for Specified Applications:**
   To check if a specific application is installed on an instance, you need to connect to the instance and execute a command that checks for the application. This can be done using the `ssm` client in boto3. Here is a sample script that checks if Docker is installed:
   ```python
   import boto3

   def check_application(instance_id, application):
       ssm = boto3.client('ssm')
       response = ssm.send_command(
           InstanceIds=[instance_id],
           DocumentName='AWS-RunShellScript',
           Parameters={'commands': ['which {}'.format(application)]},
       )
       command_id = response['Command']['CommandId']
       output = ssm.get_command_invocation(
           CommandId=command_id,
           InstanceId=instance_id,
       )
       return 'not installed' in output['StandardOutputContent']

   print(check_application('your-instance-id', 'docker'))
   ```
   This script sends a command to the instance that checks if Docker is installed and returns True if it is not installed.

4. **Iterate Over All Instances:**
   Finally, iterate over all instances and check if the application is installed on each one. Here is a sample script that checks if Docker is installed on all instances:
   ```python
   import boto3

   def list_instances():
       ec2 = boto3.resource('ec2')
       for instance in ec2.instances.all():
           if check_application(instance.id, 'docker'):
               print('Docker is not installed on instance {}'.format(instance.id))

   list_instances()
   ```
   This script prints the ID of all instances where Docker is not installed.
