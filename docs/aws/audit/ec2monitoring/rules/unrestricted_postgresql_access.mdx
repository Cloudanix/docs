---
title: Unrestricted PostgreSQL Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access to TCP port 5432 (PostgreSQL Database).

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, SOC2, GDPR, NIST
