---
title: Unrestricted Telnet Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access to TCP port 23 (Telnet).

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, SOC2, GDPR, NIST, CIS, HIPAA, ISO27001
