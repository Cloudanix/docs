### More Info:

Unused AWS EC2 key pairs should be decommissioned to follow best practices.

### Risk Level

Medium

### Address

Security

### Compliance Standards

AWSWAF

### Remediation

How to remove any unused EC2 key pairs provisioned within your AWS account?

#### Using AWS Console

1. Login to your AWS Management Console.
2. Go to the EC2 dashboard.
3. Click on the "Key Pairs" option from the left-hand side menu.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "Unused AWS EC2 Key Pairs Should Be Removed" Policy.)
4. Select the key pair that you want to delete.
5. Click on the "Actions" drop-down menu and select "Delete Key Pair".
6. Confirm the deletion by clicking on the "Delete" button.
7. Repeat steps 4-6 for any other unused key pairs that you want to delete.
