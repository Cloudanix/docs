### Remediation

How to enable SSM Parameter encryption?

#### Using AWS Console

1. Open the AWS Systems Manager console.
2. In the left navigation pane, choose Parameter Store.
3. Choose the parameter you want to encrypt.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "SSM Parameters Should Be Encrypted" Policy.)
4. Choose Edit.
5. Select the SecureString radio button.
6. Check the box next to KMS Key ID.
7. Choose the KMS key ID that you want to use for encryption.
8. Choose Save Changes.
9. Repeat steps 3-8 for all parameters that need to be encrypted.
