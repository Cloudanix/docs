---
title: Unrestricted DNS Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access to TCP and UDP port 53 (DNS).

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, APRA, MAS, NIST
