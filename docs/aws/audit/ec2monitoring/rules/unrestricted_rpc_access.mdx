---
title: Unrestricted RPC Access Should Not Be Allowed
---

### More Info:

No security group should allow unrestricted inbound access to TCP port 135 (RPC).

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, SOC2, NIST, CIS, ISO27001, HIPAA, GDPR
