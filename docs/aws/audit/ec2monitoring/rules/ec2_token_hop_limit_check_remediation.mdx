2. In the navigation pane, choose 'Your VPCs'.

3. Select the VPC you want to check and then click on 'Actions'.

4. In the dropdown menu, click on 'Edit Tenancy' and check the current setting. If it is set to 'default', it means that the EC2 instances in this VPC can be either dedicated or shared instances.    ```
