### Remediation

How to enable X-Ray tracing for Beanstalk environments?

#### Using AWS Console

1. Log in to the AWS Management Console and navigate to the Elastic Beanstalk service.
2. Select the Beanstalk environment for which you want to enable X-Ray tracing.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "Ensure X-Ray Tracing Is Enabled For Elastic Beanstalk Environments" Policy.)
3. In the left-hand menu, click on "Configuration".
4. Scroll down to the "Software" section and click on the "Edit" button.
5. In the "Environment properties" section, click on "Add another property".
6. Enter "AWS_XRAY_TRACING_ENABLED" as the key and "true" as the value.
7. Click on the "Apply" button to save the changes.
8. Once the changes are saved, redeploy the application to the Beanstalk environment.
9. Verify that X-Ray tracing is enabled by checking the AWS X-Ray console for traces from the Beanstalk environment.
