### Remediation

How to enable alert notifications for Beanstalk environments?

#### Using AWS Console

1. Log in to the AWS Management Console and navigate to the Elastic Beanstalk console.
2. Select the environment for which you want to enable alert notifications.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "Enable Alert Notifications For Elastic Beanstalk Events" Policy.)
3. Click on the "Configuration" tab in the left-hand menu.
4. Scroll down to the "Notifications" section and click on the "Edit" button.
5. In the "Notifications" dialog box, select the checkbox next to the type of notification you want to receive (e.g. "Email notifications").
6. Enter the email address(es) where you want to receive the notifications.
7. Click on the "Save" button to save the changes.
