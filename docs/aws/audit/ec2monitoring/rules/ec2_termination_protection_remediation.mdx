### Remediation

How to enable termination protection for EC2 instances?

#### Using AWS Console

1. Open the AWS Management Console and navigate to the Amazon EC2 service.
2. From the left navigation pane, click on "Instances" to view the list of EC2 instances.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "Termination Protection Should Be Enabled" Policy.)
3. Locate the EC2 instance for which you want to enable termination protection and select it by clicking on the checkbox next to it.
4. In the instance details pane at the bottom, click on the "Actions" button.
5. From the dropdown menu, select "Instance Settings" and then click on "Change Termination Protection".
6. In the "Change Termination Protection" dialog box, select the checkbox for "Enable termination protection".
7. Click on the "Apply" button to enable termination protection for the selected instance.
8. Repeat steps 3 to 7 for any additional EC2 instances you want to enable termination protection for.
