### Remediation

How to check if EC2 Instances has Multiple Security Groups?

#### Using AWS Console

1. Navigate to the EC2 dashboard in the AWS Management Console.
2. Select the EC2 instance that has multiple security groups.
    (In the Cloudanix Console, navigate to "Misconfig" page and look for Affected Assets for "EC2 Instances With Multiple Security Groups" Policy.)
3. In the bottom pane, scroll down to the "Security Groups" section.
4. Click on the "Edit" button.
5. Remove any unnecessary security groups from the list.
6. Click "Save" to apply the changes.
