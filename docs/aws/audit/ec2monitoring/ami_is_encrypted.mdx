---
title: EC2 AMIs Should Be Encrypted
---

### More Info:

Amazon Machine Images (AMIs) should be encrypted to fulfill compliance requirements for data-at-rest encryption.

### Risk Level

High

### Address

Security

### Compliance Standards

NIST, PCIDSS, ARPA, MAS, HIPAA, GDPR
