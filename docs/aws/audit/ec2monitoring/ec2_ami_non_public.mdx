---
title: EC2 AMIs Should Not Be Public
---

### More Info:

AWS AMIs should not be shared publicly with the other AWS accounts to prevent exposing sensitive data.

### Risk Level

High

### Address

Security

### Compliance Standards

NIST, ARPA, MAS
