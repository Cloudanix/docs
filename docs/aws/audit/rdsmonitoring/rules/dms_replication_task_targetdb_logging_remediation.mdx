5. Click on the parameter group name to open the "DB Parameter Groups" page. Here, look for the "logging_enabled" parameter. If the value of this parameter is set to '0', then logging is not enabled for the target DB of the DMS Replication Task.

2. Once the AWS CLI is set up, you can list all the DMS replication tasks using the following command:

   ```
   aws dms describe-replication-tasks
   ```

   This command will return a JSON output with details of all the DMS replication tasks.

3. From the output, identify the replication task for which you want to check the target DB logging status. Note down the 'TargetEndpointArn' of the replication task.

4. Now, use the 'TargetEndpointArn' to describe the target endpoint and check if logging is enabled. Use the following command:

   ```
   aws dms describe-endpoints --filters Name=endpoint-arn,Values=<TargetEndpointArn>
   ```

4. In the output, look for the 'ExtraConnectionAttributes' field. If logging is enabled, you should see an entry like 'general_log=1;slow_query_log=1'. If these entries are not present or set to 0, then logging is not enabled for the target DB.

```python
import boto3

session = boto3.Session(
    aws_access_key_id='YOUR_ACCESS_KEY',
    aws_secret_access_key='YOUR_SECRET_KEY',
    region_name='us-west-2'
)
```

2. **Create a client for RDS and DMS:**

```python
rds = session.client('rds')
dms = session.client('dms')
```

3. **Get the list of all DMS tasks and their target endpoints:**

```python
try:
    response = dms.describe_replication_tasks()
except ClientError as e:
    print(f"Error retrieving DMS tasks: {e}")
else:
    tasks = response['ReplicationTasks']
    target_endpoints = [task['TargetEndpointArn'] for task in tasks]
```

4. **Check if logging is enabled for each target endpoint:**

```python
for endpoint_arn in target_endpoints:
    try:
        response = rds.describe_db_instances(DBInstanceIdentifier=endpoint_arn)
    except ClientError as e:
        print(f"Error retrieving DB instance: {e}")
    else:
        db_instance = response['DBInstances'][0]
        if db_instance['Engine'] == 'mysql':
            if 'LogExports' in db_instance and 'general' in db_instance['LogExports']:
                print(f"Logging is enabled for {endpoint_arn}")
            else:
                print(f"Logging is NOT enabled for {endpoint_arn}")
```

This script will print out whether logging is enabled for each DMS target endpoint. If logging is not enabled, it will print a message indicating so. 

Please replace `'YOUR_ACCESS_KEY'` and `'YOUR_SECRET_KEY'` with your actual AWS access key and secret key. Also, replace `'us-west-2'` with the actual region you are working in.
