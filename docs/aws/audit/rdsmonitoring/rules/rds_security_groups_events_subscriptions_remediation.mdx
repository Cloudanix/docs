#### Using CLI

1. Install and configure AWS CLI: Before you can start using AWS CLI, you need to install it on your local machine and configure it with your AWS account credentials. You can do this by running the following commands:

   ```
   pip install awscli
   aws configure
   ```
   You will be prompted to provide your AWS Access Key ID, Secret Access Key, default region name, and default output format.

2. List all RDS instances: Use the following command to list all your RDS instances:

   ```
   aws rds describe-db-instances --query 'DBInstances[*].[DBInstanceIdentifier]'
   ```
   This command will return a list of all your RDS instances.

3. Check the event subscription for each RDS instance: For each RDS instance, you can check the event subscription by running the following command:

   ```
   aws rds describe-event-subscriptions --subscription-name <subscription-name>
   ```
   Replace `<subscription-name>` with the name of your event subscription. This command will return information about the event subscription, including whether it is enabled or not.

4. Analyze the output: If the `Enabled` field in the output is `true`, then the event subscription is enabled. If it is `false`, then the event subscription is not enabled. If the command does not return any output, then the event subscription does not exist.
