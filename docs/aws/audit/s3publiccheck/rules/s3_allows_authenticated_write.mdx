---
title: S3 Bucket Should Not Allow WRITE Access to Authenticated Users
---

### More Info:

S3 buckets should not allow WRITE access to AWS authenticated users through S3 ACLs.

### Risk Level

Medium

### Address

Security

### Compliance Standards

PCIDSS, APRA, MAS, NIST
