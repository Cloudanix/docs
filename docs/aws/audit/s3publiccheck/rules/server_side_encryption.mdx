---
title: S3 Buckets Should Enforce Server Side Encryption
---

### More Info:

AWS S3 buckets should protect their sensitive data at rest by enforcing Server-Side Encryption (SSE).

### Risk Level

High

### Address

Security

### Compliance Standards

SOC2, NIST, GDPR, HIPAA
